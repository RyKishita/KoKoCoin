# コイン

## 追加方針

- シーズン1では禁止
  - ランダム
    - このゲームは運要素が強い。さらに要素を加えないため。
    - ターン数を見る効果は実質ランダム
      - 本当は外したいが、コイン数が大きく減るので断念
  - 自身のライフ回復
    - そんな事より攻撃する方が良いようなバランスにする。
    - 相手も回復する可能性があるなら許可
  - 相手の手持ちコインを破壊
    - そんな事より攻撃する方が良いようなバランスにする。
  - 相手の設置攻撃コインを破壊
    - 設置攻撃コインは安定したダメージ元としたい。
    - あるとしても手持ちコイン全て捨てるぐらいのコストは必要
  - 相手の行動阻害
    - そんな事より攻撃する方が良いようなバランスにする。
    - 現状でも状態異常があるが、おまけ程度の効果にしている。
- シーズン2では何でもありの方針
- シーズン3では全コイン刷新
  - 加えてオンライン対戦の基本ルールはシーズン3のコインのみ使用可とする予定
- 優先
  - 遊びの幅を広げるもの
    - 特殊勝利など
  - コイン単体よりは複数の組み合わせで強力になるもの

- ドロー
  - サポートコインで、ストックまたは捨てコインから指定のコインを取得する
    - タグやコイン種類だと選択があるので保留
  - 使用すると相手がコイン〇枚をドロー
    - ライブラリアウトデッキ。ルールがストック空で敗北の場合

  - 設置時にカウンターを〇個持つ。このコインが捨てコインとなる時、カウンターがあるなら代わりに減らす。
- 妨害
  - 相手がサポートコインを使用したとき、手持ちコインのこのコインを捨てコインにすることで効果を無効にする
    - 現状サポートコイン対策が無い
    - 使うかどうかの選択を出すと止まるので無くしたい
- その他
  - 手持ちコインに〇コインがある場合、このコインと合体して〇コイン（デュエル中のみ）になる
    - ファイターなど、効果を持たないコインのペアになるコイン。合体すると、エンチャンテッド・ファイターになるなど
    - 合体したコインがさらに合体の効果を持っいて、どんどん違うコインになる
  - 設置コイン。ターン終了ごとに〇コインステータスを取得
  - ダメージを与えた場合、手持ちコインに残る/戻る
    - 複製を得る、の効果とほぼ同じ。これは直接攻撃コインで実装済み

  - ？があるかぎり、このコインは捨てコインにならない
    - 魔術師の幻影シリーズが効果としては逆だが実装済み
